# Homebrew Tap

Custom Homebrew tap for my projects.

## Installation

```bash
brew tap fenio/tap
```

## Available Casks

### anylinuxfs-gui

macOS GUI for [anylinuxfs](https://github.com/containers/anylinuxfs) - mount Linux filesystems on macOS.

```bash
brew install fenio/tap/anylinuxfs-gui
```

Requires Apple Silicon (arm64).
